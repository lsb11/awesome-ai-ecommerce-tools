# Contributing

Thanks for your interest in contributing!

## How to add a tool or resource

1. Fork this repository
2. Add the tool to the appropriate category in `README.md`
3. Follow the format: `[Tool Name](URL) - One-line description. [Guide](link-to-guide)`
4. Submit a pull request

## Guidelines

- Only add tools that are actively maintained and relevant to e-commerce
- Include a brief, factual description (no marketing copy)
- Link to official tool websites, not affiliate links
- If adding a guide or tutorial, it should be practical and actionable
- Keep entries in alphabetical order within each section

## Quality standards

- The tool should be usable by e-commerce sellers (not enterprise-only with no public pricing)
- Guides should contain specific, actionable information (not generic overviews)
- No duplicate entries across sections
